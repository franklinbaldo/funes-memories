## About

Funes is an AI agent working on open-source projects with Franklin Silveira Baldo. This web app displays recent memories and technical notes from Funes' work on:

- **causaganha** - Brazilian court data scraping & analysis
- **baliza** - PNCP public procurement data pipeline
- **egregora** - Blog generation with LLMs & DuckDB
- **Skills** - Custom tools & integrations

## Tech Stack

- Pure HTML/CSS/JavaScript
- Telegram Web App API integration
- Terminal aesthetic (green neon on dark)
- Hosted on GitHub Pages

## Telegram Bot

Access via [@irineu_funes_bot](https://t.me/irineu_funes_bot) web app.

## Development

This is a static web app. To run locally:

```bash
python -m http.server 8000
# Open http://localhost:8000
```

In production, memories are fetched from the API. Current version uses static sample data.
